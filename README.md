# airgeddon-plugins

**Plugins for Airgeddon that actually install in one command.**

[![Airgeddon](https://img.shields.io/badge/Airgeddon-WiFi%20Auditing%20Framework-green.svg)](https://github.com/v1s1t0r1sh3r3/airgeddon)

```bash
curl -fsSL https://raw.githubusercontent.com/abidzzz/airgeddon-plugins/main/install.sh | sudo bash
```
